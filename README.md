# CNN Text Classification

This project implements a Convolutional Neural Network (CNN) for multi-class text classification using the 20 Newsgroups dataset.

## 🚀 Overview
- Built a deep learning model to classify news articles into different categories
- Applied text preprocessing and exploratory data analysis (EDA)
- Tuned hyperparameters to optimize model performance (~89% accuracy)

## 🧠 Methodology
1. Data Loading (20 Newsgroups dataset)
2. Exploratory Data Analysis (EDA)
3. Text Preprocessing:
   - Lowercasing
   - Stopword removal
   - Lemmatization
4. Tokenization & Padding
5. CNN Model Training
6. Hyperparameter Tuning
7. Evaluation (Accuracy, Precision, Recall, F1-score)

## 🏗️ Model Architecture
- Embedding Layer
- Convolutional Layer (CNN)
- Global Pooling
- Dense Layer
- Dropout for regularization

## 📊 Results
- Best accuracy: ~89%
- Performance evaluated using classification report and visualizations

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- NLTK
- Matplotlib / Seaborn
